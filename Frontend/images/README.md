# Images Directory

This directory contains images used in the Study Pro Global website.

## Required Images

The following images are referenced in the website but are optional (the site has fallback handling):

1. **admission.svg** - Illustration for admission assistance section
2. **interview.svg** - Illustration for interview preparation section
3. **payment.svg** - Illustration for tuition fee payment section
4. **visa.svg** - Illustration for visa documentation section
5. **support.svg** - Illustration for post-arrival support section

## Image Specifications

- Format: SVG, PNG, or JPEG
- Recommended size: 500x500 pixels minimum
- Style: Modern, professional illustrations
- Color scheme: Should complement the primary colors (#2563eb blue and #10b981 green)

## Adding Images

To add images:
1. Place the image files in this directory
2. Ensure file names match those referenced in index.html
3. For best results, use SVG format for scalability

## Notes

- The website includes `onerror` handling, so if images are not available, they will be hidden gracefully
- All images should be optimized for web to ensure fast loading
